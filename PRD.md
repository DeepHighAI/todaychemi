## 1. 제품 한 줄 정의

> **오늘케미** — *"보기전에 미리 합을 맞춰 볼까?"* — 한국·동남아 25-45 직장인을 위한 **관계 사주 CRM 노트앱**. 단일 핵심 = 본인 사주 ⊕ 인연(별명·생년월일·성별)의 만세력 + 명리 해석.

- **카테고리**: 관계 사주 CRM
- **무드**: Notion × Apple Notes — 따뜻하고 모던한 노트앱 (한지·먹·전통 무게 ❌)
- **플랫폼**: PWA + TWA Android 1차 → iOS Phase 4 검토

### 비협상 제약

1. **인연 입력은 별명만** — 라벨도 "이름" 금지, "별명"으로 통일
2. **자유 입력 AI 채팅 화면 없음** — 모든 LLM 결과는 *구조화 카드/리포트* 안에서만
3. **한지·먹·전통 사주집 무드 금지** — 그래픽 메타포 = 그라디언트 메쉬 + 절제된 라인
4. **별자리·MBTI·타로 시각 요소 금지**
5. **운세 단정 톤 금지** — "반드시 / 100% / 확실히" 카피 금지. 모든 케미카드 하단에 면책 의무
6. **전문가 실시간 상담·UGC 커뮤니티 게시판 없음** (ADR-020)

## 3. 정보구조

### 3.1 메인 네비 (3탭 + 이런건어때 시트)

```
┌────────────────────────────────────────┐
│   ✨ 진단·리포트 이런건어때 (탭바 상단 시트)   │
├────────────────────────────────────────┤
│           (선택된 탭 콘텐츠)               │
├────────────────────────────────────────┤
│  [오늘]      [케미피드]      [내 프로필]          │
└────────────────────────────────────────┘
```

| 탭 | 핵심 컴포넌트 | 우선 설계 |
|---|---|---|
| **오늘** | 오늘 케미, 진행 중 해석, 최근 메모 | 오늘 홈 |
| **케미피드** (핵심) | 인연 그리드, 카드 탭 → 합보기 + 타임라인 | 인연 그리드, 디테일, 케미카드, 타임라인 |
| **내 프로필** | 본명식, 일주 해석, 오행 차트, 운세 흐름 | 본명식, 오행맵 |
| **이런건어때 시트** | 신규 진단, 추천 리포트, 친구합 (Pair Quiz) | 시트 1·2단계 |

### 3.2 정보 위계
1. **핵심**: 합보기 + 관계 진화 타임라인 — 모든 화면이 이쪽으로 유입
2. **보조**: 마이플레이 / 딥합 / 친구합 (Pair Quiz) / 시즌 큐레이션 — 이런건어때 시트와 빈 슬롯에서만 노출
3. **무료**: 오늘 케미 / 본명식 무제한 조회 — retention 표면

> 이런건어때 시트가 메인 탭과 동급으로 보이지 않게 시각 우선순위 조절. 핵심(인연 탭)이 가장 무겁게.

## 4. 핵심 사용자 플로우

### Flow A — 신규 가입자 첫 5분 (Phase 1, 인연 등록 우선 + 2트랙 분기 + 6단계 입력 순서, ADR-032·029)

> **⚠ Amended 2026-06-13 (절충안, FGI §16 ADR-032·029 Amend)**: 아래 원안 순서는 기각. **현행** = 가입 → 본인 4-step 온보딩 → 오늘홈. 인연 등록 유도 = ① 오늘홈 인연 0건 hero(ADR-032 카피 + 등록 CTA 주역) ② `/relations/new` 생일 단계 "생일을 잘 몰라요" → "처음 보는 나"(whatif `first`) Track B 분기. 원안은 설계 사료로 보존.

**입구 화면 설명 카피 의무 노출 (ADR-032 보강)**: *"먼저 떠오르는 사람을 고르면, 그 사람과의 합흐름을 보기 위해 내 생년월일도 함께 계산합니다."* — 사용자가 "내 정보도 안 넣었는데 왜 남의 생일을 먼저?"라고 느끼지 않도록 본인 정보 사용 의도를 명시.

**입력 6단계 (ADR-032 보강)**:

```
1. 오늘 떠오르는 사람 선택  ("+ 인연 등록" 진입)
2. 별명 입력  (실명 금지 카피)
3. "생일을 아나요?" 예 / 아니오 분기
4-A. 예 → 상대 생년월일 + 시 입력  (Track A)
4-B. 아니오 → "첫만남 플레이"  (Track B)
5. 내 생년월일·시 입력  (짧은 인라인 단계, 첫 진입 시만)
6. 결과 생성  → 케미카드 [1][2][4] 즉시 인지
```

이후:
- Google OAuth는 1단계 *직전*에 1회. 1회 무료 해석 + D+1 케미 다시 맞추기 진입 토큰 자동 지급 토스트
- (Track A) 케미카드 → 스크롤 후 [6] 푸터에서 D+1 알림 예약 / 공유 / 메모 (2차 액션, ADR-026)
- (Track B) 자기 성찰 카드 → 첫 만남 TIP 3 + 메모 → "인연을 알게 되면 정확 해석 받기" CTA로 Track A 유도
- 두 번째 해석부터 결제 안내 — 무료 부적(10부적) 우선 차감, 부족 시 ₩1,000 즉시결제 (ADR-039 pay-per-use)
- 인연 등록은 2명까지 무료, 3번째부터 ₩1,000/10부적 — mode 단계 사전 고지 + 402 시 결제 시트 (ADR-039 §9, 2026-06-10)

### Flow B — 일상 리텐션 (Phase 1)
1. 매일 아침 푸시 → 오늘의 케미카드
2. 어제 해석 인연 D+1 알림 → "[다시 보기 — 6부적/₩600]" CTA
3. 매주 일요일 → 주간 흐름 리포트 추천

### Flow C — 결정 지원 (Phase 1)
1. 이런건어때 시트 → 합보기 + 상황 컨텍스트 selector 진입
2. 양쪽 사주 결합 해석 → 케미카드 + 상황 맞춤 일단이거해봐 3
3. 합메모 → 타임라인 기록

### Flow D — 관계 진화 회고 (Phase 1.5)
1. D+7 푸시 → 인연 디테일 → 타임라인 뷰
2. 케미 다시 맞추기(6부적/₩600) → 합흐름 그래프
3. 메모 추가 → D+30 알림 예약

### Flow E — 위젯 (Phase 2)
- iOS/Android 홈스크린 위젯 → 1탭 시 *마지막 해석 인연* 빠른 케미 다시 맞추기 또는 신규 등록

### 6.2 컴포넌트별 시각 결정

| # | 컴포넌트 | 결정 사항 |
|---|---|---|
| 1 | 일주 chip | 60갑자 컬러 매핑 = 5개 오행 컬러를 천간 음양으로 라이트/다크 ±10% 명도 변형. 컴포넌트 라이브러리 1개로 60종 자동 생성 |
| 2 | 합게이지 | 원형 + 4구간 컬러 그라디언트. 정수, 4구간 라벨 KR/EN i18n |

### 6.3 6모드별 본문 변형 (ADR-025)

각 모드는 [4] 본문 3섹션 *제목·구성*만 변형. 1·2·3·5·6·7·8은 모드 무관 동일.

| 모드 | 본문 3섹션 변형 |
|---|---|
| 일합 | 호환성 / 강점 / 주의점 / 협업 팁 3 |
| 친구합 | 합점수 / 우정 유형 / 거리 조정 팁 |
| 돈합 | 합점수 / 파트너십 적합도 / 계약 체크포인트 3 |
| 첫합 | 합점수 / 진척 가능성 / 다음 만남 가이드 |
| **썸합** | 합점수 / 감정 흐름 / 진척 가능성 / 다음 만남 가이드 |
| **오래합** | 호환성 / 관계 안정도 / 갈등 패턴 / 장기 추천 액션 |

> 모드 선택 화면(S-06)에 *3단계 가이드 카피* 의무 노출: ① **"소개팅 = 처음 만나는 단계"** ② **"썸 = 만난 적 있고 관계 정의 모호"** ③ **"오래합 = 교제 시작 후"**. 사용자가 자기 분류를 정확히 할 수 있도록 각 모드 옆 한 줄 카피 의무.

> *모르는사람 모드는 ADR-025로 제거*. 동일 사용 맥락(첫 미팅·새 부서)은 §13.2 마이플레이 "첫만남 플레이"(8부적/₩800, 본인 사주 단독 자기 성찰형)로 흡수. S-08 이런건어때 시트 진입.

## 12. 6모드 명리적 관점 (ADR-025)

각 모드별 명리 강조 포인트는 케미카드 [5] 근거 보기 영역에 시각화되며, 문항 카피의 *암시 톤*에도 반영.

| 모드 | 명리 강조 |
|---|---|
| 일합 | 십신(정관·편관·식상) — 권위·실행 |
| 친구합 | 일지 합·반합 + 오행 보완 — 정서 동조 |
| 돈합 | 재성·인성 + 시지 — 결정·자산 |
| 첫합 | 천간합·지지합 첫 발현 + 도화·홍염 |
| **썸합** | 도화·홍염살 + 천간합·지지합 첫 발현 + 충·반합(감정 변동) — 초기 설렘 |
| **오래합** | 합·삼합·충극(장기 균형) + 오행 장기 균형 + 십성 안정도 — 관계 정의 명확 |

> 명리 용어를 *번역하지 않고* 그대로 노출(브랜드 신뢰 시그널). 단, 첫 등장 시 짧은 ⓘ 툴팁 한 줄(S-08-T).

> *모르는사람 모드는 ADR-025로 제거*되었으며, 동일 맥락은 마이플레이 "첫만남 플레이"(8부적/₩800, 본인 사주 단독)로 흡수.
