"월률분야 테이블" 출처로 명기됨. 일관성 문제 없음.

**가중치 10:5:3 = 자체 단순화 (방향 정합 · 서열 일부 이견).** 三命通會 사령일수
(`classic_smth_006`, `classic_smth_007`)와 대조:

- 정기 비중: 고전은 한 달 30일 중 18~23일(60~77%) — 구현 10/18(55.6%)은 **정기 우위 방향은
  정합하나 고전 일수보다 보수적**.
- 왕월(子午卯酉·정기 23일·중기/여기 없음 또는 7일): 구현(정기10·중기/여기 해당 없음) 정합.
- **[권고 R1]** 사계월(辰戌丑未)에서 고전은 여기 7일 > 중기(묘고) 5일인데 구현은 중기5 > 여기3로
  **서열 역전**. 점수 무관(LLM 근거 분포에만 영향)이나 고전 정합도를 높이려면 사계월 한정
  여기/중기 가중 교환 또는 일수 비례 가중 채택 검토. → **산식 변경 = 별도 컨펌 + derived_version 2
  + 프롬프트 버전 범프 동반 필요** (ADR-040 cache-coupling rule).
- **[권고 R2]** (선택) 정기 가중을 일수 비례(약 18~23/30)로 상향하면 고전 정합도 ↑ — R1과 동일
  절차 필요. 현행 10:5:3도 "정기 우위" 취지는 보존하므로 유지 가능.

## 항목 2 — 지지 십신 = 정기 기준 단순화 → PASS (현대 통설 + 고전 간접 근거)

- 子平真詮 「論用神」 "以日乾配月令地支，而生克不同，格局分焉"(`classic_zpjj_001`) — 격국 판별이
  월지(지지)와 일간의 생극 관계로 이뤄지며, 실무 통변은 지지 본기(정기) 중심.
- 사령일수의 정기 압도(18~23일/30일, `classic_smth_006/007`)가 정기 대표성의 정량 근거.
- 중기·여기를 십신 판별에서 제외한 것은 단순화로 §6.1에 명기되어 있음 — 유지.

## 항목 3 — 신강약 억부 점수제 → 구조 PASS (고전 근거) · 수치는 자체 단순화

- **득령 최대 가중(+20)**: 淵海子平 「繼善篇」 "欲知貴賤，先觀月令乃提綱"(`classic_yhzp_008`) +
  子平真詮 "八字用神，專求月令"(`classic_zpjj_001`) — **월령 최우선 = 고전 직접 근거** ✓
- **일간 중심 판정**: 繼善篇 "專用日干為主本" ✓
- **강약→억부 대응**: 滴天髓 「衰旺論」 "旺則宜洩宜傷，衰則喜幫喜助"(`classic_jcs_006`) ✓
- **계수(20/10/8/8/±15)·임계(70/30)**: 고전에 수치 없음 — **자체 단순화(명기 유지)**. 검증 불가
  영역이므로 §6.2 "잠정" 단서는 "수치 한정 잠정"으로 유지.
- **단정 회피 정합**: 滴天髓 "旺中有衰者存，不可損也"(`classic_jcs_007`) — 단순 점수제의 한계를
  고전이 직접 경고. 구현은 LLM에 verdict 문자열만 전달(숫자 점수 미전달)하고 UI에 점수를 노출하지
  않으므로 **경고 취지와 정합**. 환각 가드("제공 필드 외 단정 금지")도 동일 방향.

## 항목 4 — 용신·희신 억부 1차 룰 → PASS (고전 근거 + 현대 통설 선택 명기)

- **억부 원칙**: 滴天髓 「體用」 "要在扶之抑之得其宜"(`classic_jcs_009`) + 「衰旺論」
  (`classic_jcs_006`) — 부억 = 고전 직접 근거 ✓
- **신약→인성 primary·비겁 secondary**: 고전 "衰則喜幫喜助"는 幫(비겁)·助(인성)을 병렬 — 인성
  고정 우선은 **현대 통설 범위 내 선택**(생조 우선). §6.3에 선택 사유 명기됨 — 유지.
- **신강→[관성·식상·재성] 중 최강**: "旺則宜洩宜傷"(洩=식상)·"抑之"(관성) — 정합.
- **중화→최소 오행 보완**: 滴天髓 「中和論」 "能識中和之正理"(`classic_jcs_008`) — 중화 지향 근거.
  보완 룰 자체는 자체 단순화(명기).
- **한계 2건 명기 유지**: ① 조후 미반영(궁통보감 계열 자산과 분리) ② 종격(從格) 미지원 —
  기존 자산 `classic_jcs_005` "從強從弱，順其勢可"가 종격을 직접 언급하므로, 극단 명식에서
  억부 후보가 빗나갈 수 있음. **"용신 후보" 라벨(확정 아님)이 이미 적용되어 있어 수용 가능**.

## 항목 5 — 궁위 4라벨 → PASS (고전 근거 있음, 4/4)

三命通會 「論年月日時」(`classic_smth_008`, `classic_smth_009`) 대조:

| 구현 라벨 | 고전 근거 | 판정 |
|---|---|---|
| 년주 = 뿌리·초년 | 年為本 / 年為根 | ✓ 직접 근거 |
| 월주 = 사회·부모 | 月為兄弟僚友(형제·동료=사회축) | ✓ 사회축 직접 근거. '부모'는 통설(月為父母) — 三命通會 이 구절은 年為父로 학파 차 존재. 절충 라벨로 수용 |
| 일주 = 배우자궁·자아 | 日為主為妻為己身 | ✓✓ 완전 정합 |
| 시주 = 미래·자식 | 時為子孫 / 時為實 | ✓✓ 완전 정합 |

"배우자궁·자아" 라벨이 친구합·일합 모드에도 노출되는 점(의뢰서 질문 5)은 고전이 궁위를 모드
무관 고정 개념으로 다루므로 **단일 라벨 유지 타당** — LLM이 모드 맥락에서 해석을 조절한다.

## 항목 6 — 연령차 밴드 → 검증 불가 (명기)

동갑/1-3/4-6/7+ 구간은 명리 고전의 영역이 아님(문화·제품 설계 선택). 고전 검증 대상 아님으로
판정하고 §6.6의 ±1 오차 단서 유지.

## 항목 7 — salient 문장 톤 → PASS

"상대 일간(己) = 내 일간 기준 정재(재성)" 류는 십신 판별 규칙의 결정적 사실 서술로 고전 통변
관행과 동일 차원. 강약 관련 단정은 verdict 문자열에 한정되고(항목 3) 환각 가드가 미제공 주제
언급을 차단하므로 안전.
